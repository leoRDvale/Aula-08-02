# Aula-08-02

Atividade: Sistema de Configuração de Notebook com Fluent Interface (Builder Moderno) e Inner Class.

Descrição do Problema:

Você está desenvolvendo um sistema de montagem personalizada de notebooks para uma loja online. Os clientes podem selecionar componentes específicos para montar seu notebook ideal.

O sistema deve permitir a construção de um notebook com os seguintes componentes:

- Processador
- Memória RAM
- Unidade de Armazenamento
- Placa de Vídeo
- Sistema Operacional

Para garantir que os notebooks criados sejam objetos imutáveis (não podem ser modificados após criados), você deve utilizar o Builder com Fluent Interface como inner class da classe Notebook.
